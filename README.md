## ﻿Rock Paper Scissors Game
This is the part of Odin Project in JavaScript Foundation course. I build the game of "Rock Paper Scissors" by using knowledge and concepts within the Odin Project's course.

## How to play
1. Open 'index.html'.
2. Enter your move (Rock, Paper, or Scissors) on the pop-up prompt.

## How it works
1. Uses 'Math.random()' to generate computer move.
2. Uses 'toLowerCase()' to make sure that user input isn't case-sensitive.
3. Uses 'playGame()' to build loop of 5-round game.

## What i learn
- The use of 'for' loop for playing game multiple rounds. I feel it is a bit tricky at first, but still manage to work it through.
- The use of 'if/else' logic to map game rule's condition.
